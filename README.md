# 抽牌系統

一個用於故事創作、角色設定的抽卡網頁應用，支援四種卡牌類型（人物、關係、個性、劇情），可自訂卡池、查看牌陣，並切換淺色／深色主題。

---

## 功能特色

### 抽卡頁面
- **四種卡牌**：人物卡、關係卡、個性卡、劇情卡
- **抽卡規則**：
  - 人物卡、個性卡：抽 3 張 → 翻開揭曉 → 去掉 1 張，保留 2 張
  - 關係卡：抽 2 張 → 翻開揭曉 → 去掉 1 張，保留 1 張
  - 劇情卡：共抽 4 張，全部保留
- 單張抽卡、翻牌揭曉、選牌丟棄
- 全部清除重抽
- 抽完四種卡後可開啟「查看牌陣」

### 牌陣
- 依指定格式展示 9 張牌（2 人物、2 個性、1 關係、4 劇情）
- 紅框：人物卡 | 黃框：個性卡 | 綠框：關係卡 | 紫框：劇情卡
- 同性質的卡可點選兩張交換位置

### 設定頁面
- 自訂四種卡池（每行一張卡牌名稱）
- 套用設定
- 恢復預設卡池
- 保存設定（勾選後寫入 localStorage，下次開啟自動載入）

### 主題
- **淺色**：馬卡龍淺紫 + 馬卡龍色系 + 白（預設）
- **深色**：莫蘭迪藍 + 水藍，霧蒙水墨感

## 使用方式

1. **直接開啟**：在瀏覽器中開啟 `index.html` 即可使用
2. **抽卡流程**：
   - 選擇卡牌類型（人物／關係／個性／劇情）
   - 點「抽一張」依規則抽滿
   - 點「翻開揭曉」揭露牌面
   - （人物、關係、個性）點選要去掉的牌，再點「去掉一張」
3. **牌陣**：四種卡皆完成後，點「查看牌陣」，可點選同性質牌交換位置
4. **設定**：進入「設定抽卡參數」頁，編輯卡池並套用

## 預設卡池

- **人物卡**：約 240 張（職業、學生、身份等）
- **關係卡**：約 200 張（師徒、戀人、對手、契約關係等）
- **個性卡**：約 245 張（開朗、冷靜、傲嬌、犬系、貓系等）
- **劇情卡**：約 350 張（台詞、情節、結局等）
